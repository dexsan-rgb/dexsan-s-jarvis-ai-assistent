# JARVIS AI Assistant 🚀

An advanced, light-weight, and custom voice assistant with hybrid capabilities.

## Key Features ⚡
- **Double Clap Wakeup:** Wakes up from deep sleep on detecting a double clap.
- **Hybrid AI Core:** Connects to Hugging Face or falls back to DuckDuckGo/Wikipedia scraping to summarize answers instantly.
- **System Commands:** Open Arduino IDE, VirtualBox, Chrome, and play YouTube videos.
